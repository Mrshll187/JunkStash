# JunkStash Webapp
* Java 8
* Spark Framework 2.5
* Spring 4.3.2
* Mongo 3.0.0
* Angular 1.5.8
* Font Awesome 4.6.3
* Bootstrap 3.3.7
* UI-Bootstrap 2.1.2
* jQuery 3.1.0
* JQuery-UI 1.12.0

# To Run
* mongod --fork --dbpath <USER_PATH> --logpath <USER_PATH>
* mvn clean install
* java -jar <USER_PATH>/junkStash.jar <PORT> (defaults 4567)
* Open browser to : localhost:<PORT>
